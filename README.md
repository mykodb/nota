# Nota

## 项目简介
`Nota` 是一款基于 Minecraft 的 Fabric 模组。
它是作者有趣的想法，也是作者在学习编程和模组开发过程中的一次尝试。

## 如何使用
`Fabric API` `Fabric Language Kotlin`
